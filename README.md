5) Примечания по WebSocket

Для реализации чата используется публичный echo WebSocket сервер - wss://ws.ifelse.io

Сервер не является полноценным чат-ботом и возвращает обратно (echo) то же сообщение, которое отправил клиент. Полученный ответ отображается в интерфейсе как сообщение «ассистента», что соответствует требованиям тестового задания.

Использование публичного WebSocket сервиса накладывает ряд ожидаемых ограничений:

-соединение может периодически закрываться

-сервер может быть временно недоступен

-возможны ограничения на количество одновременных подключений

В процессе разработки наблюдались onclose события без явной ошибки, а также нестабильность соединения при обновлении страницы. Эти случаи обрабатываются логированием и не приводят к нарушению работы интерфейса.

Также было выявлено различие поведения в зависимости от браузера и окружения:

Microsoft Edge — WebSocket соединение устанавливалось и работало стабильно

Google Chrome — в отдельных случаях соединение не устанавливалось или закрывалось сразу после открытия

Предполагаемой причиной являются ограничения сетевого окружения, антивирусного ПО, firewall, корпоративных прокси или расширений браузера, а также возможные блокировки доступа к публичным echo WebSocket серверам.

Данное поведение является ожидаемым при использовании внешнего WebSocket сервиса без SLA и не связано с логикой приложения.
